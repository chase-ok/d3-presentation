Data Driven Documents

Why should we care?

Some very cool visualizations:

  • 512 Paths
  • Wind Speeds
  • Airplane Delays
  • Remittances
  • Racial Divide

What is it?

D3.js is a JavaScript library for manipulating documents based on data... D3 allows you to bind arbitrary data to a Document Object Model (DOM), and then apply data-driven transformations to the document.

d3js.org

What is it really?

Collection of tools for producing many different kinds of visualizations

  • Loading external data
  • Maps and topography
  • HTML/SVG/CSS manipulation
  • Scales: linear, log, color, ordinal...
  • Layouts: networks, axes, histograms, trees, packs, bundles...
  • Interactivity
  • Animations
  • Math

How does it work?

  • Links your data to HTML/SVG elements
  • Edits your webpage on-the-fly
  • Executes transformations based on user input

Usage Warning

Explanatory vs. Exploratory

What D3 is NOT good for

  • Basic plots that you can make in one line with matplotlib
  • Supporting older browsers
  • Data secrecy

What D3 is really GREAT for

  • Visualizing multi-dimensional data
  • Interactivity
  • Geographical data
